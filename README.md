# MoodSync - Digital Wellbeing Assessment Platform

MoodSync е веб-платформа за евалуација на дигиталната и менталната благосостојба кај младите. Апликацијата овозможува следење и анализа на афективната состојба, квалитетот на сон и нивото на перципиран психосоматски стрес кај младата популација.

Преку имплементација на компаративни извештаи директно инспирирани од методологијата и наодите на меѓународната **HBSC (Health Behaviour in School-aged Children)** студија, системот нуди динамички анализи и корисни насоки за стабилизација на когнитивниот, емоционалниот и психосоматскиот баланс во периодот на адолесценцијата (11-15 години).

## Карактеристики
- **Дневни показатели (Твојот биланс):** Квантитативна проценка и индекс на благосостојба од 0 до 100 прилагоден за младите.
- **Динамички HBSC извештаи:** Системот во реално време ја проценува когнитивната стабилност и дава релевантен контекст базиран на HBSC параметрите.
- **Историја на записи:** Преглед на претходно внесените состојби со кориснички забелешки за контекстот на училиштата и секојдневниот живот.

---
1. Инсталација на зависности (Dependencies)
Пред првото стартување, потребно е да ги инсталирате потребните пакети со извршување на следнава команда во терминалот:

Bash
npm install

2. Стартување на развојниот сервер
За да ја пуштите апликацијата во локален развоен режим, извршете ја командата:

Bash
npm run dev
